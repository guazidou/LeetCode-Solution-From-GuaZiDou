# 问题描述

反转字符串

题目连接: [https://leetcode.com/problems/reverse-string/](https://leetcode.com/problems/reverse-string/)



# Python {#python}

使用python字符串强大的切片解决

```py
class Solution(object):
    def reverseString(self, s):
        """
        :type s: str
        :rtype: str
        """
        return s[::-1]
```
